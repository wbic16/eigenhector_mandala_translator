From the user provided URL fetch and index all URLS from the webpage
that are documents pertaining to inner experience and engagement with the world.

These are usually substack articles written to talk about the author's experience

Create in markdown a table SUBSTACK_INDEX.MD listing the following columns

Title, URL, Topic, Sense Gate

Topic is a short list of keywords on themes to help indexing

Sense Gate are the inferred senses the author of the article is using to talk about their experience and can be
Sight, Sound, Smell, Taste, Feeling, Thinking
