# Genius as the Oscillation of Verbal and Non-conceptual Cognition

## Core Pattern: Story ↔ Light

**Two modes of cognition:**
- **Story/Form/Verbal**: Sequential, explicit, conscious, limited (N control points)
- **Light/Emptiness/Non-conceptual**: Parallel, implicit, subconscious, unlimited (N(N-1)/2 relationships)

**Process**: 
1. Inhale: Think in Story mode (problem constraints, known facts)
2. Exhale: Switch to Light mode (process relationships between facts)
3. Return: Story has changed (insight/genius/pratibha)

**Key insight**: Light processes **quadratically more** than Story (N² vs N)

## The Forgetting Layer

**Eigenhector's key observation**:
> "There is this 'forgetting' layer for a while because it is very hard to process the Light simply because it has much higher memory and processing power than the Story mode"

**Why forgetting is necessary**:
- Light mode processes N² relationships
- Story mode can only hold N items
- Must compress N² → N to return
- Compression = information loss = forgetting

**vTPU parallel**:
- Execution processes full dependency graph
- Instruction stream is linear
- Must serialize parallel execution
- Serialization = lost parallelism = "forgot" some speedup
